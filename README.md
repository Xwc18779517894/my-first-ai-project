# AI-TODO：智能任务清单（带 AI 总结与鼓励）

一个用 **Python + Flask + DeepSeek API** 快速构建的网页任务管理小工具。  
支持添加/删除任务，一键让 AI 分析当天计划并给出中文鼓励话语。

## 项目演示

![任务清单截图](https://github.com/Xwc18779517894/my-first-ai-project/raw/main/screenshots/demo.png)


## 技术栈

- 后端：Python + Flask  
- AI 集成：DeepSeek API（deepseek-chat 模型）  
- 前端：Tailwind CSS CDN + 自定义样式  
- 开发工具：Cursor（AI 辅助编码）

## 快速启动

1. 克隆仓库  
   ```bash
   git clone https://github.com/Xwc18779517894/my-first-ai-project.git
   cd my-first-ai-project

2.安装依赖

     Bashpip install -r requirements.txt
3.替换 API Key（重要！）

     打开 app.py，找到 headers = {"Authorization": "Bearer sk-xxx"} 这行，把 sk-xxx 替换成你自己的 DeepSeek API Key（从 https://platform.deepseek.com 获取）。千万不要把 Key 上传到 GitHub！
4.运行项目Bashpython app.py

     浏览器访问打开 http://127.0.0.1:5000 即可看到任务清单界面。
